= Excluindo uma cópia Snapshot
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Você pode excluir uma cópia Snapshot em um sistema de armazenamento usando o `snapdrive snap delete` comando.
