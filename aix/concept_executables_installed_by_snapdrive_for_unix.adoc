= Executáveis instalados pelo SnapDrive para UNIX
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
O SnapDrive para UNIX instala executáveis para o sistema operacional AIX.

Estes executáveis são instalados em `/opt/NetApp/snapdrive/bin/snapdrive`.
