= Multipathing no SnapDrive para UNIX
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
O SnapDrive para UNIX suporta multipathing. O SnapDrive para UNIX não suporta o uso de aliases para dispositivos, criados usando os `mknod` comandos e/ou `rendev`.
